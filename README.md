# SyntaxWear - E-commerce de Tênis e Sneakers

## Descrição
Este projeto é um protótipo de e-commerce focado na venda de tênis e sneakers, chamado "SyntaxWear". Desenvolvido com HTML e CSS, ele apresenta uma interface moderna e responsiva, com seções dedicadas a categorias de produtos, destaques, formulário de newsletter e navegação intuitiva.

## Funcionalidades
*   **Página Inicial:** Layout atraente com banner principal (Hero Section) e chamada para ação.
*   **Navegação:** Menu superior para categorias principais (Masculino, Feminino, Outlet) e links secundários (Nossas Lojas, Sobre, Minha Conta, Ajuda, Carrinho).
*   **Categorias de Produtos:** Seção visualmente organizada para explorar diferentes estilos de tênis (Casual, Esporte, Moderno, Futurista).
*   **Grid de Produtos:** Área de destaque para exibir produtos com cards informativos.
*   **Rodapé Completo:** Contém links de navegação por categoria, formulário de inscrição para newsletter e links para redes sociais.
*   **Responsividade:** Design adaptável para proporcionar uma boa experiência em diferentes tamanhos de tela (desktop e mobile).

## Tecnologias Utilizadas
*   **HTML5:** Para a estrutura e conteúdo da página.
*   **CSS3:** Para o estilo, layout e responsividade do site.

## Estrutura do Projeto
O projeto está organizado da seguinte forma:

```
ecommerce-syntaxwear/
├── index.html                  # Página principal do site
├── README.md                   # Este arquivo
├── css/                        # Contém todos os arquivos CSS
│   ├── base/                   # Estilos base e utilitários
│   │   ├── reset.css           # Reset de estilos padrão do navegador
│   │   ├── style.css           # Estilos globais
│   │   └── variables.css       # Variáveis CSS
│   ├── components/             # Estilos para componentes específicos
│   │   ├── footer.css
│   │   ├── header.css
│   │   ├── hero.css
│   │   ├── product-category.css
│   │   └── product-grid.css
│   └── layout/                 # Estilos para o layout geral (pode estar vazio ou com estilos específicos de layout)
│       └── footer.css          # Exemplo de arquivo de layout para o footer (atualmente vazio)
└── images/                     # Contém todas as imagens e ícones do projeto
    ├── icons/                  # Ícones utilizados
    ├── logo/                   # Logo da SyntaxWear
    └── (outras imagens de produtos/banners)
```

## Como Usar
Para visualizar e interagir com o site:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/ecommerce-syntaxwear.git
    ```
    (Substitua `https://github.com/seu-usuario/ecommerce-syntaxwear.git` pelo link correto do seu repositório, se aplicável).

2.  **Navegue até o diretório do projeto:**
    ```bash
    cd ecommerce-syntaxwear
    ```

3.  **Abra o arquivo `index.html`:**
    Basta abrir o arquivo `index.html` em seu navegador web preferido (Chrome, Firefox, Edge, etc.).
